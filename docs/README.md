# Supplementary documentation

## Figure assets

Batch preview images for the main project `README.md` are stored in **`assets/`** at the repository root (for example, `train_batch1.jpg`, `train_batch2.jpg`, `val_batch1.jpg`, `val_batch2.jpg`, alongside `workflow.jpg`).

Typical sources on the training workstation are Ultralytics run outputs (e.g. `runs/segment/<run_name>/train_batch*.jpg`, `val_batch*_pred.jpg`), copied or renamed to match the filenames referenced in `README.md`.
